# Cryptography

Encoding, hashing, and encryption — theory and real-world attack scenarios.

## Structure
- `week1.py` — Base64, hex encoding, MD5/SHA256 hashing, salting
- `week2.py` — Hash cracking, identifying hash types, weak crypto
- `week3.py` — AES/RSA encryption, common misconfigurations
- `week4.py` — JWT attacks, algorithm confusion, none algorithm

## Key Concepts
- Encoding vs Encryption vs Hashing
- Avalanche effect
- Rainbow table attacks and salting
- Real world Base64 cookie vulnerabilities
